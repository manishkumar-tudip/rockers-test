To run this app, you need to take few steps and also make sure you have Node and Mongodb up and running.

1. Take the source code to you machine (either git clone or the zip)
2. Navigate to the directory in your cli and run "npm install" and then "npm start"
3. Adde few brands and cloths in your mongo collection.
4. In your browser navigate to url: http://localhost:3000/static/  
5. Here you would get an interface to search an see the results on the same page.
